class Player{
    フィールド変数
        int step;//歩数を格納する
        
    メソッド
        ゴール地点を探す{
            迷路情報からサーチ
        }

        ゴールまでのルート探索{
            左右優先でゴールまで移動，壁があったらルートに戻るまで壁に沿って歩く
            (最悪ルートに戻れなくてもゴールできればok)
            ゴールと同じ横座標に移動できれば，次は縦に移動する。
            これをゴールできるまで繰り返す。
        }

        ルート書き込み{
            決まったルートをMazeの迷路情報に書き込み
        }
}
